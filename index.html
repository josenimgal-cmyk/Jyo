One-time setup — takes about 3 minutes. This connects your app to a free Firebase database so both of you see the same data on any device, in real time.
Step 1 — Create Project

Go to console.firebase.google.com, sign in with Google, click Add project, name it anything (e.g. joseph-jyot), and finish the wizard.

Step 2 — Enable Realtime Database

In your project, click Realtime Database in the left sidebar → Create database → pick any location → select Start in test mode → click Enable.

Step 3 — Get your Database URL

After enabling, you'll see a URL at the top of the Realtime Database page that looks like https://joseph-jyot-default-rtdb.firebaseio.com. Copy it and paste it below.
